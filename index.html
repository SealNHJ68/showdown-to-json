Instrucciones para usar la aplicación:
  • Crea tu equipo en formato showdown (puede ser un Team, Box o todo un Backup), si no lo has creado, crealo aquí
  • Pega tu formato en el primer recuadro y da clic en el botón "Cargar equipo."
  • Copia y usa a tu gusto el formato JSON generado!
  • Nota: Trata de usar formato válido, esta aplicación no válida si el formato es correcto, usar un formato alterado puede mostrar información errónea.
